# 3D-Game-Project
Group project for CS 530 

Name of Game: Resident

Group Members: Gabriel Sawaya, Rafael Aguilar, Juan Conriquez, Skyler Varney

Description: 
You wake up in a mysterious house, confused as to how you ended up there, you roam around only to find 
a stranger trying to keep you inside the house. Your goal is to scape the house by exploring the house,
exploring the different puzzles there is in order to unlock the front door and scape. 

Current target for game: 

- Create basic game map (haunted house-style)
- Implement basic FPS movement for the player and enemy AI behavior
- Create enemy model with animations
- Implement audio and visuals for effects

Goals for each person:


Gabe: Programmer (Gameplay & Systems)
- Implement player movement, camera, and interaction

- Create enemy AI (patrol, chase, attack)

- Implement inventory system (keys, ammo, simple UI)

- Connect all gameplay elements and fix core bugs



Juan: Level Designer (Environment & Triggers)
- Greybox the entire map (rooms, hallways, layout)

- Replace greybox with final assets and props

- Add lighting, fog, and atmospheric effects

- Place enemies, item pickups, and puzzle elements

- Set up triggers (doors, scares, music cues)



Skyler: Artist (3D Assets, Animations, Visual Polish)
- Collect/create 3D assets (rooms, props, items)

- Prepare character/enemy models (using premade rigs if needed)

- Import and set up animations (walking, attacking, idle)

- Create UI visuals (menu, health, inventory icons)

- Add VFX (blood, smoke, particles)



Rafael : Audio/Narrative/QA (Sound, Story, Testing)
- Gather and implement sound effects (footsteps, doors, ambience)

- Add horror audio layers (creaks, whispers, music stingers)

- Write short story elements (notes, simple intro, objectives)

- Perform regular testing and bug reporting

- Balance gameplay (enemy speed, ammo, difficulty)
